# AI Rumsvisualisering (Next.js + Vercel)

## Snabbstart
1) Skapa en **hemlig nyckel** på https://platform.openai.com/api-keys
2) På **Vercel → Project → Settings → Environment Variables**: lägg `OPENAI_API_KEY` med din nyckel.
3) **Importera** detta repo från GitHub till Vercel och **Deploy**.

## Lokalt
```bash
npm install
npm run dev
```
Öppna http://localhost:3000
